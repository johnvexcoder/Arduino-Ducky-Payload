# Arduino-Ducky-Payload Library for the [USB Rubber Ducky](https://hak5.org/products/usb-rubber-ducky?variant=39874478932081) by [Hak5](https://hak5.org)

**This repository provides a collection of Arduino-based payloads designed for BADUSB devices, offering a flexible platform for custom automation and security testing scripts. To utilize these payloads, users must compile and flash the code using the [Arduino IDE](https://www.arduino.cc/en/software), ensuring proper configuration for their specific hardware. Please note that johnvexcoder does NOT guarantee payload functionality, as successful execution depends on hardware compatibility, system configurations, and the specific environment in which the device is deployed. Users are encouraged to test thoroughly in controlled settings before any practical application.**
